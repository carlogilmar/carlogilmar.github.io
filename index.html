Create a simple groovy script Include the next imports import io.vertx.groovy.core.Vertx import io.vertx.groovy.ext.mongo.MongoClient Add the configuration for Mongo DB. def config = Vertx.currentContext().config() def uri = config.mongo_uri if (uri == null) { uri = "mongodb://localhost:27017" } def db = config.mongo_db if (db == null) { db = "test"//<---------------database name } def mongoconfig = [ connection_string:uri, db_name:db ] def mongoClient = MongoClient.createShared(vertx, mongoconfig) Add some data in the script.
